接続方法
LEDの長い方（＋）を１３番ピン、短い方（ー）をGNDに

コード

delayの後の()の数字で時間を変えられる
1000で1秒、単位はms


void setup() {
  // put your setup code here, to run once:
  pinMode(13,OUTPUT);
}

void loop() {
  // put your main code here, to run repeatedly:
  digitalWrite(13,HIGH);
  delay(500);
  digitalWrite(13,LOW);
  delay(500);
}

暗いと点く、明るいと消える(CdSセルと発光ダイオード)

接続方法
これとこれ

コード

(あとで調整)って書いてあるところは、
シリアルモニタの数値を見て調整必要！
数値は人によって変わる


void setup() {
  pinMode(13, OUTPUT);
  Serial.begin(9600); // 追加（確認用）
}

void loop() {
  int value = analogRead(A0); // ← 光センサー読む
  Serial.println(value);      // ← 数値確認

  if (value > 500) {          // ← ここがポイント（あとで調整）
    digitalWrite(13, HIGH);   // 暗い → 点灯
  } else {
    digitalWrite(13, LOW);    // 明るい → 消灯
  }

  delay(200);
}
